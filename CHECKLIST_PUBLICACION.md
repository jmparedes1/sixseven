# Checklist final de publicación

## Antes de subir

- [ ] `index.html` está en la raíz.
- [ ] `app.js` está en la raíz.
- [ ] `style.css` está en la raíz.
- [ ] `assets/` está en la raíz.
- [ ] `firebase-config.js` tiene el proyecto correcto.
- [ ] `firebase.rules.json` está incluido.

## Firebase

- [ ] Authentication anónima activada.
- [ ] Realtime Database creado.
- [ ] Reglas copiadas desde `firebase.rules.json`.
- [ ] Reglas publicadas.

## Prueba mínima

- [ ] Abre la pantalla principal.
- [ ] Pulsa **Crear evento**.
- [ ] Crea evento con el código general.
- [ ] Comprueba que aparece código y QR.
- [ ] Entra desde otro navegador/incógnito con el código.
- [ ] Vota con dos usuarios distintos.
- [ ] Comprueba que aparece el match y el chat.
- [ ] Entra como administrador.
- [ ] Cambia el estilo visual y pulsa guardar.
- [ ] Comprueba que se mantiene.
