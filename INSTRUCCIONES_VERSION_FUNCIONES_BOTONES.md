# INSTRUCCIONES PARA SUBIR ESTA VERSIÓN

1. Descomprime el ZIP.
2. Sube TODO el contenido directamente a la raíz del repositorio de GitHub Pages.
3. En la raíz deben quedar:
   - index.html
   - app.js
   - style.css
   - firebase-config.js
   - firebase.rules.json
   - assets/
   - diagnostico-funciones.html

4. No subas la carpeta contenedora completa.
5. En Firebase:
   - Authentication > Sign-in method > Anonymous: activado.
   - Realtime Database > Rules: pega el contenido de firebase.rules.json y pulsa Publicar.

6. Abre:
   https://jmparedes1.github.io/sixseven/?v=botones-final

7. Pulsa Ctrl + F5.

8. Diagnóstico:
   https://jmparedes1.github.io/sixseven/diagnostico-funciones.html

9. Prueba recomendada:
   - Crear evento con código general.
   - Escribir nombre del evento.
   - Escribir clave de creador de mínimo 6 caracteres.
   - Repetir clave.
   - Crear evento.
   - Entrar desde incógnito con el código creado.
